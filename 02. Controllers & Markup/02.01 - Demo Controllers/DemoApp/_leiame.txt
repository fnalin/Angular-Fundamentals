** Instruções deste projeto **

1. Criar uma blank solution
2. Criar um projeto C# / Web / Empty
3. Instalar os pacotes baixo:
	- bootstrap
	- AngularJS.Core
4. Criar pasta app (Nossa aplicação FrontEnd estará aqui)
5. Criar o arquivo app\EventDetails.html
6. Criar o arquivo app\app.js (Nosso Módulo)
7. Criar a pasta app\controllers
8. Criar o Controller da página EventDetails: app\controllers\EventController.js
9. Instalar o pacote p/ sanitize do HTML e referenciá-lo na página:
	- AngularJS.Sanitize
10. Add a dependência ngSanitize no módulo
